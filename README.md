# u2-python-lesson2-assessment
This repository contains materials for U2 Coding with Python – Lesson 2: Assessment.

This assessment covers assessment criteria:-

2.1 Create a GitHub repository

2.2 Create a fork GitHub repository

2.3 Demonstrate the use of commits for version control.  Learners must: 

• Use commit descriptive messages 

• Use commits to refactor 

• Push commits to a remote repository 

• Edit and delete commits 

• Undo or erase a commit 


3.1 Demonstrate how to verify a remote using GitHub.  

3.2 Demonstrate how to clone remote repositories using GitHub. 

3.3 Demonstrate how to push files to own remote repository using GitHub
